# Flavorlestown
# [Watch the demo on YouTube](https://youtu.be/5BllzclAKxs)


Do you find Flavourtown too flavourful or astheatically pleasing. Well i have have a solution for you. Just install this exetesnion and make this town just as flavourful as you like.

##Features, how it works and how to use it?
- Move the slides to change the flavourfulnes of the website

How to install? 
- download the .crx file
- go to chrome and type in to the search bar ://extensions it works also on some other browsers than chrome like brave.
- Enable Developer Mode by clicking the toggle switch next to the Developer mode text
- drag and drop the dowbloaded .crx file on top of the ://extensions site
- open the exetension in the exetension menu the icon looks like a puzle piece.
